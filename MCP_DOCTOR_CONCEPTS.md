# MCP Doctor Concepts

## What MCP Doctor Is
A diagnostic substrate for AI systems that explains authority, rejection, and conflict using persisted evidence.

## What MCP Doctor Is Not
- Not an agent
- Not a UI
- Not a policy engine
- Not a RAG framework

## Core Concepts
- Authority evidence
- Acceptance and rejection
- Conflicts
- Temporal validity

## Medical Records Metaphor
Findings are immutable diagnostic records, not logs.

## Scope
MCP Doctor reasons about why knowledge was or was not allowed to influence an answer.

## Outputs
- Deterministic views
- UIResource-compatible representations

## Constraints
MCP Doctor never embeds raw evidence in client-facing outputs.
